# Intrusion Prevention System (IPS) 

## Graduation Project

### Overview
This repository contains the source code and documentation for our graduation project: a network **Intrusion Prevention System (IPS)**. The system is designed to monitor network traffic in real-time, detect malicious activities based on predefined signatures and anomaly detection, and actively block threats to secure the network infrastructure.

### Key Features
* **Real-time Traffic Monitoring:** Analyzes packets flowing through the network.
* **Threat Detection:** Identifies potential attacks (DoS, DDoS, Port Scanning, etc.).
* **Active Prevention:** Automatically blocks malicious IP addresses or drops harmful packets.
* **Logging & Reporting:** detailed logs of detected threats and system actions.
* **Dashboard:** (Optional: Add if you have a UI) User-friendly interface to view alerts and statistics.

### Technologies Used
* **Programming Language:** Python / C++ (Edit based on your project)
* **Libraries:** Scapy, Pandas, NumPy (Edit based on your project)
* **Database:** SQLite / MySQL
* **Tools:** Wireshark, Nmap (for testing)
